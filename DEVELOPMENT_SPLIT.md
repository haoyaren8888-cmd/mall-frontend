# 开发分工

当前只记录开发任务，不包含课程设计报告和答辩材料。

| 成员 | 占比 | 前端开发内容 |
| --- | ---: | --- |
| 组长 | 70% | 项目骨架、路由、布局、登录注册、用户状态、核心购物流程、结算、支付、订单页整合、接口联调 |
| 组员A | 10% | 首页、分类页、商品列表、商品详情、商品卡片组件、分类缓存 |
| 组员B | 10% | 购物车页面、购物车条目组件、本地购物车、地址管理、结算页地址选择 |
| 组员C | 10% | 后台布局、仪表盘、商品管理、分类管理、订单管理 |

建议分支：

- 组长：`feature/core-flow`
- 组员A：`feature/product-catalog`
- 组员B：`feature/cart-address`
- 组员C：`feature/admin-manage`
