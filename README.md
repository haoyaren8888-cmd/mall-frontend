# 太原理工大学校园闲置商城前端

Vue3 + Vite + Element Plus + Pinia + Axios。

## 启动

```bash
npm install
npm run dev
```

默认访问：`http://localhost:5173`

后端接口默认地址：`http://localhost:8080/api`。如果后端地址变化，可以设置环境变量 `VITE_API_BASE_URL`。

## 默认账号

- 管理员：`admin / 123456`
- 学生用户：`user / 123456`

## 主要页面

- 前台：首页、分类筛选、闲置详情、发布闲置、我的发布、意向清单、交易确认、模拟支付、交易记录、联系地址
- 后台：闲置商品审核、分类管理、交易订单管理
